# 发票批量重命名工具设计文档

## 1. 需求概述
用户需要批量重命名PDF发票文件，根据Excel表格中的信息（如发票号码、金额、公司名称等）进行重命名。要求：
- 在Windows 7及更高版本上运行（兼容Win7、Win8、Win10、Win11）
- 操作人员会基本电脑操作，可以点击按钮
- 重命名格式可以灵活组合，根据Excel表头列自由定义
- 可能有Excel表格，也可能没有
- 发票文件为PDF格式
- 发票号码通常在文件名中，特别是在"DZFP"后面的数字里
- 如果文件名中没有发票号码，需要从PDF文件内容中读取发票号码进行匹配

## 2. 核心功能
### 2.1 文件管理
- 选择包含发票PDF文件的目录
- 可选选择Excel文件（.xlsx格式）
- 显示目录中的PDF文件列表

### 2.2 Excel数据读取
- 读取Excel文件，获取表头和所有行数据
- 支持.xlsx格式（使用openpyxl库）
- 显示Excel内容预览

### 2.3 重命名格式定义
- 提供格式定义界面，用户可以：
  - 选择要使用的列（通过复选框或下拉菜单）
  - 定义列的顺序
  - 添加分隔符（如下划线、短横线等）
  - 实时预览重命名结果
- 格式语法：`{列名1}_{列名2}_{列名3}.pdf`
- 支持自由组合，如：`{序号}-{货物或应税劳务名称}-{金额}-{公司}-{销方名称}.pdf`

### 2.4 匹配逻辑
**场景A：有Excel表格**
- 需要建立Excel行与PDF文件的对应关系
- **主要匹配方式**：根据Excel中的"发票号码"列与PDF文件名进行匹配
  - **文件名提取规则**：
    1. 优先查找"DZFP"后面的数字（如：DZFP12345678.pdf → 12345678）
    2. 如果没有"DZFP"，尝试提取文件名中的数字序列
    3. 支持多种分隔符（下划线、短横线、空格等）
  - 与Excel中的发票号码列进行匹配
  - 匹配成功则建立对应关系
- **备选匹配方式**（当文件名中没有发票号码时）：
  1. 从PDF文件内容中提取发票号码（使用PDF解析库）
  2. 按文件修改时间顺序匹配
  3. 按文件创建时间顺序匹配
  4. 手动匹配（拖拽或下拉选择）
- 用户可选择匹配方式，系统默认使用发票号码匹配

**场景B：没有Excel表格**
- 需要从PDF文件名或内容中提取信息
- 简单方案：基于原始文件名进行简单处理（添加序号、前缀等）
- 复杂方案：需要OCR识别PDF内容（不推荐，增加复杂度）

### 2.5 预览与确认
- 显示重命名预览表格：原始文件名 → 新文件名
- 允许用户调整或取消某些文件的重命名
- 确认后执行批量重命名

### 2.6 安全机制
- 操作前备份原始文件名映射
- 提供撤销功能（可选）
- 防止文件名冲突（自动添加序号）

## 3. 技术方案
### 3.1 技术栈
- Python 3.8（兼容Windows 7）
- GUI：Tkinter（标准库，无需额外安装）
- Excel处理：openpyxl
- 打包：PyInstaller（生成exe文件）

### 3.2 架构设计
```
invoice_renamer/
├── main.py              # 主程序入口
├── gui/                 # 图形界面模块
│   ├── __init__.py
│   ├── main_window.py   # 主窗口
│   ├── file_selector.py # 文件选择组件
│   ├── excel_viewer.py  # Excel预览组件
│   ├── format_builder.py# 格式构建器
│   └── preview_table.py # 预览表格

│   ├── __init__.py

│   ├── file_matcher.py  # 文件匹配
│   ├── renamer.py       # 重命名逻辑

```

### 3.3 用户界面设计
1. **主界面**：
   - 顶部：文件夹选择区域
   - 中部左侧：Excel文件选择和预览
   - 中部右侧：PDF文件列表
   - 底部：格式定义区域和预览表格

2. **格式定义界面**：
   - 列选择：复选框列表，显示所有Excel列名
   - 顺序调整：上移/下移按钮
   - 分隔符选择：下拉菜单（下划线、短横线、空格等）
   - 实时预览：显示示例文件名

3. **操作流程**：
   - 选择文件夹 → 选择Excel（可选）→ 定义格式 → 预览 → 确认重命名

## 5. 用户操作流程
1. 双击打开程序
2. 点击"选择文件夹"按钮，选择包含发票的文件夹
3. 如果有Excel文件，点击"选择Excel文件"按钮
4. 在格式定义区域，勾选要使用的列，调整顺序
5. 选择分隔符
6. 查看重命名预览
7. 确认无误后，点击"开始重命名"按钮
8. 等待完成，查看结果

## 6. 注意事项
1. **文件名安全**：移除非法字符（\ / : * ? " < > |）
2. **重名处理**：自动添加序号避免覆盖
3. **编码问题**：确保支持中文文件名
4. **性能考虑**：处理大量文件时的响应性
5. **错误恢复**：提供操作日志和撤销功能
