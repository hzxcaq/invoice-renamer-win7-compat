## 3. 技术实现细节
### 3.1 Excel读取
- 使用openpyxl库
- 支持.xlsx格式
- 读取表头和所有行数据
- 返回字典列表

### 3.2 文件匹配
- 优先从文件名中提取发票号码
  - 查找"DZFP"后面的数字
  - 支持多种分隔符
- 备选：从PDF内容中提取发票号码
  - 使用PyPDF2或pdfplumber
  - 查找发票号码相关文本
- 匹配算法：
  1. 遍历Excel行
  2. 对每个PDF文件尝试提取发票号码
  3. 与Excel中的发票号码列匹配
  4. 建立对应关系

### 3.3 格式定义
- 格式语法：`{列名1}_{列名2}_{列名3}.pdf`
- 支持自由组合
- 实时预览示例

### 3.4 安全重命名
- 移除非法字符（\ / : * ? " < > |）
- 处理重名（自动添加序号）
- 支持中文文件名

## 4. 依赖库
- openpyxl：Excel读取
- PyPDF2：PDF内容提取（可选）
- pdfplumber：PDF内容提取（可选，更强大）
- tkinter：图形界面（标准库）
